Vérifier les réponses des IA médicales contre les référentiels français.

Un assistant IA répond à une question clinique. Le serveur MCP Bayes vérifie automatiquement la réponse contre les recommandations de la HAS et les décisions de l'ANSM, puis la corrige si nécessaire.

Le serveur MCP Bayes est open source et s'intègre à n'importe quel assistant IA, qu'il s'agisse de Claude, ChatGPT, Gemini ou d'un agent maison. Une opportunité pour les pouvoirs publics : garantir que les IA déjà utilisées par les citoyens respectent les référentiels du pays.
